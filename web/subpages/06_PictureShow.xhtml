PictureShow Application

Overview

PictureShow is a Java Swing based app using basic features of swing "window" and "component" classes.
The app main functionalities are customly built on top of those.

You can download the Desktop version of PictureShow samples below in case of security problem with webstart (JNLP).
And here is the PictureShow source code on GitHub to dig deeper into it.

Some highlights from the custom features (see detailed list at the bottom Appendix):
  1. Animated, ordered, multi-level, groupped, autosizing / auto-positioning behaviour
  2. Show / hide button, background image
  3. Design layouts: add / remove / resize / move pictures, group of pictures, backgrounds.
  4. Manage layouts: new / load / save / delete layouts.
The Java core app also manages integration elements to give a fullscale solution:
  1. MySQL database behind the scenes to store and handle PictureShow layouts
  2. Storage for pictures on a webserver or local path (both works automatically based on installation)
  3. Can be run from a usual webpage using JNLP

What PictureShow can be used for?

It's a real mini presentation tool (not only a project introducing me as a programmer).
A flexible app to design layouts to show, animate, activate etc. pictures on different level of layers.
Telling stories, roadmaps, family albums, map based journeys etc.
